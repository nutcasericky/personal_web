What is it about

A secret sharing scheme is a method of protecting a secret among a group of participants(n) in such a way that only certain specified subsets of the participants(t) (those belonging to an access structure) can reconstruct the secret. A secret sharing scheme is normally initialized by a trusted dealer who securely transfers a piece of information related to the secret, called a share, to each participant in the scheme. When t number of participants come together they can recover the secret. Shamir Secret Sharing Scheme(SSS) makes use of polynomial f(x) with higheset degree t-1 which can be reconstructed using Langrange's Interpolation formula when t number of points on the graph of f(x) are provided.

Implementation

I have implemented SSS with Python, which can be tried out below. The source code can be found on the link at the bottom of the home page. The more detailed explanation of the implementation of the scheme is as followed:

Set up phase

1. The dealer is asked to input total number of participants n, number of participants required to recover the secret s(s is smaller than n), the secret s and a prime number that is greater than n and s.
2. A polynomial f(x) is generated with highest degree t-1 and its coefficients belong to Z_p and s the value of the original secret as the constant term.
3. n randomly chosen values belonging to Z_p will be substituted into the polynomials as x values and evaluate corresponding y values. Then shares (x and y) will be sent individually to each participant.

Reconstruction phase

1. A minimum of t number of shares is needed. Then, using Lagrange's Interpolation formula, we can interpolate the value of y at x = 0 which is the secret.
